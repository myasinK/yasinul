Lift-off!

First post - favorite shortcuts

Posted by YK
On October 1, 2018

For my first post I wanted to keep things simple, but useful! Going forward I'll use this space predominantly to document useful tips, techniques, etc. as I continue to delve deeper into software development.

So for this post, I wanted to put up some useful keyboard shortcuts for Visual Studio.

Getting comfortable with any IDE necessitates the knowledge of relevant keyboard shortcuts. While I've only scratched the surface of what's possible without the use of my mouse/touchpad I can already notice the increase in coding speed and efficiency made possible by the shortcuts I know.

Like any modern IDE or text-editor, Visual Studio makes it possible to perform just about any UI function with the press of 2 or 3 key-strokes. If you're already a coding jedi, you probably don't use the mouse much (and you're probably not reading this in the first place). But if you're training to become one, like I am, it's hard to miss the wealth of information available online. I'll mention a couple that I found super useful.

visualstudioshortcuts.com is super handy and comprehensive, albeit a little overwhelming. But it's great to have bookmarked for future reference.

The second source I'd mention is the mothership itself - the Microsoft Developer Network. Needless to say, one would be hard-pressed to find a more comprehensive guide to the subject than what's available on MSDN.

In case you're curious about how to change or set keyboard shortcuts in Visual Studio, that's pretty easy also. Simply navigate to Visual Studio >> Tools >> Options >> Environment >> Keyboard. In fact, there's a shortcut for that - Alt + Space . That'll open the Options window and at that point you can use Tab to navigate to the different fields and look for the function you're looking to modify, etc.

But for the beinners out there, and to occasionally jog my own memory I'll put up a short list of shortcuts that I use often. And don't try to memorize them - just use the shortcuts a few times and challenge yourself forego the mouse for a few functions. Muscle memory will do all the work! And that's advice that the great Mosh gives also.
