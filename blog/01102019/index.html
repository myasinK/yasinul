Aggregating Live Event Info for PDX

Posted by YK
On January 1, 2019

I've just put up a simple python/web-scraping project borne out of a desire to create a centralized repository of live events happening in Portland. It's merely a skeleton of what really want to do with it, but I'll leave it as is for now. I really want to finish a couple other projects I've been pretty excited about.

Needless to say, the list of venues is far from exhaustive. But I did manage to write a script that goes through schedules for Portland5, Aladdin theater, Revolution Hall, and Wonder Ballroom. Portland5 is the collective of Arlene Schnitzer Concert Hall, Newmark Theatre, Brunish Theatre, Winningstad Theatre and the Keller Auditorium.

I implemented the scraper with BeautifulSoup4 and Selenium. In hindsight, I maybe should've used Selenium for the entire project... In any case, the code is here. I did use Visual Studio, hence the proliferation of .sln files. My script is going to go through all the event pages and collect the event names, locations, and dates. It's a tiny amount of data so I'm saving it all as a JSON object.
